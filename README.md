# Appointment Interface with Vue
This is an interface made with VueJs that let you handle different Appointments

## Getting started:

### Prerequisites
  * Nodejs
  * NPM or Yarn

### Project setup

from the command line run:
```
yarn install
```
or
```
npm install
```

### Compiles and hot-reloads for development
from the command line run:
```
yarn serve
```
or
```
npm serve
```
